# Food Spectroscopy Benchmark

Comparative benchmarking framework for food spectroscopy regression using
chemometric methods and lightweight deep learning models.

## Included model families
- PLSR
- SVR
- Ridge
- ANN
- 1D-CNN

## Features
- leakage-safe nested cross-validation
- learning curve analysis
- DL complexity analysis
- automatic export of tables and figures
